# 数据示例目录

此目录存放 CSV 格式的数据示例文件。

## 文件规范

1. **编码**：UTF-8
2. **分隔符**：逗号
3. **表头**：第一行必须包含表头
4. **命名**：`{domain}_{entity}_{type}.csv`

## 数据分类

### 销售数据 (sales)
- `sales_order_sample.csv` - 订单数据示例
- `sales_product_sample.csv` - 产品数据示例

### 用户数据 (user)
- `user_profile_sample.csv` - 用户画像示例
- `user_behavior_sample.csv` - 用户行为示例

### 业务数据 (business)
- `business_knowledge_sample.csv` - 业务知识示例
- `business_metric_sample.csv` - 业务指标示例
