# 12. 多租户与文件空间设计

## 12.1 架构概览

本设计将 **OpenClaw Gateway** 与 **文件管理规范** 及 **DeepAgents 存储架构** 融合，实现基于工作空间的多租户隔离。

**设计原则**:
- **无权限验证**: 本设计不包含 JWT、API Key 等权限验证机制，仅通过 tenant_id 进行工作空间隔离
- **路径即隔离**: 通过文件路径前缀实现租户隔离，不依赖数据库 RLS 或应用层权限检查
- **DeepAgents 原生**: 复用 DeepAgents 的 CompositeBackend 和 Namespace 机制

## 12.3 核心组件设计

### 12.3.1 TenantContext (租户上下文)

```python
# tenant/context.py
import contextvars
from dataclasses import dataclass
from typing import Optional
from pathlib import Path

# 请求级租户上下文 - 每个 async 任务独立
tenant_ctx: contextvars.ContextVar[Optional['TenantContext']] = contextvars.ContextVar(
    'tenant_ctx', 
    default=None
)

@dataclass(frozen=True)
class TenantContext:
    """
    不可变租户上下文 - 贯穿整个请求生命周期
    
    来源:
    1. HTTP Header: X-Tenant-ID
    2. Session Key: tenant:{tenantId}:agent:{agentId}:{mainKey}
    """
    tenant_id: str
    tenant_type: str  # "public" | "user_public" | "user_private"
    
    # 可选的次级作用域
    session_id: Optional[str] = None
    task_id: Optional[str] = None
    
    @property
    def public_path(self) -> Path:
        return Path("public")
    
    @property
    def user_public_path(self) -> Path:
        return Path(f"{self.tenant_id}_public")
    
    @property
    def user_private_path(self) -> Path:
        return Path(f"{self.tenant_id}_private") / ".datacloud" / "workspaces"
    
    def get_workspace_path(self, level: str) -> Path:
        """获取指定层级的工作区路径"""
        if level == "app":
            return self.public_path
        elif level == "user":
            if self.tenant_type == "public":
                return self.public_path
            elif self.tenant_type == "user_public":
                return self.user_public_path
            else:
                return self.user_private_path
        elif level == "session":
            if not self.session_id:
                raise ValueError("session_id required for SESSION level")
            user_path = self.get_workspace_path("user")
            return user_path / f"session-{self.session_id}"
        elif level == "task":
            if not self.task_id:
                raise ValueError("task_id required for TASK level")
            session_path = self.get_workspace_path("session")
            return session_path / "tasks" / f"task-{self.task_id}"
        
        raise ValueError(f"Unknown workspace level: {level}")
    
    # ContextVar 操作
    def scoped(self):
        """设置上下文，返回 token 用于重置"""
        return tenant_ctx.set(self)
    
    @staticmethod
    def get() -> Optional['TenantContext']:
        """获取当前上下文"""
        return tenant_ctx.get()
    
    @staticmethod
    def reset(token):
        """重置上下文"""
        tenant_ctx.reset(token)
```

### 12.3.2 TenantResolution (租户解析)

```python
# tenant/resolver.py
from fastapi import Request

class TenantResolver:
    """
    租户解析器 - 从请求中提取租户信息
    
    注意: 本实现不包含权限验证，仅通过 tenant_id 进行工作空间隔离
    """
    
    @staticmethod
    def from_headers(request: Request) -> TenantContext:
        """从 HTTP headers 解析租户"""
        tenant_id = request.headers.get("X-Tenant-ID") or request.headers.get("x-tenant-id")
        if not tenant_id:
            # 默认使用 "default" 租户
            tenant_id = "default"
        
        return TenantResolver._build_context(tenant_id)
    
    @staticmethod
    def from_session_key(session_key: str) -> TenantContext:
        """从 session_key 解析租户
        Format: tenant:{tenantId}:agent:{agentId}:{mainKey}
        """
        parts = session_key.split(":")
        if len(parts) >= 2 and parts[0] == "tenant":
            tenant_id = parts[1]
            return TenantResolver._build_context(tenant_id)
        
        # 旧格式兼容: agent:{agentId}:{mainKey}
        return TenantResolver._build_context("default")
    
    @staticmethod
    def _build_context(tenant_id: str, tenant_type: str = "user_private") -> TenantContext:
        """构建租户上下文"""
        return TenantContext(
            tenant_id=tenant_id,
            tenant_type=tenant_type
        )
```

### 12.3.3 TenantWorkspaceManager (租户工作区管理器)

```python
# workspace/manager.py
from pathlib import Path

class TenantWorkspaceManager:
    """
    管理租户的文件空间，符合文件管理规范
    
    目录结构:
    - public/                          # 应用级共享
    - {tenantId}_public/               # 租户级共享  
    - {tenantId}_private/.datacloud/workspaces/  # 租户私有
    """
    
    def __init__(self, base_path: Path):
        self.base_path = Path(base_path)
    
    def get_workspace_path(self, context: TenantContext, level: str) -> Path:
        """
        获取指定层级的工作区路径
        
        Args:
            context: 租户上下文
            level: 工作区层级 (app | user | session | task)
        """
        if level == "app":
            return self.base_path / "public"
        
        elif level == "user":
            if context.tenant_type == "public":
                return self.base_path / "public"
            elif context.tenant_type == "user_public":
                return self.base_path / f"{context.tenant_id}_public"
            else:
                return self.base_path / f"{context.tenant_id}_private" / ".datacloud" / "workspaces"
        
        elif level == "session":
            if not context.session_id:
                raise ValueError("session_id required for SESSION level")
            
            user_path = self.get_workspace_path(context, "user")
            return user_path / f"session-{context.session_id}"
        
        elif level == "task":
            if not context.task_id:
                raise ValueError("task_id required for TASK level")
            
            session_path = self.get_workspace_path(context, "session")
            return session_path / "tasks" / f"task-{context.task_id}"
        
        raise ValueError(f"Unknown workspace level: {level}")
    
    def ensure_workspace_structure(self, context: TenantContext) -> None:
        """确保租户的工作区目录结构存在"""
        if context.tenant_type == "user_private" and context.session_id:
            # 创建完整的四层结构
            session_path = self.get_workspace_path(context, "session")
            (session_path / "skills").mkdir(parents=True, exist_ok=True)
            (session_path / "cache").mkdir(parents=True, exist_ok=True)
            (session_path / "tasks").mkdir(parents=True, exist_ok=True)
    
    def ensure_session_structure(self, tenant_id: str, session_id: str) -> Path:
        """确保会话目录结构存在"""
        session_path = (
            self.base_path / f"{tenant_id}_private" / ".datacloud" / "workspaces" /
            f"session-{session_id}"
        )
        (session_path / "skills").mkdir(parents=True, exist_ok=True)
        (session_path / "cache").mkdir(parents=True, exist_ok=True)
        (session_path / "tasks").mkdir(parents=True, exist_ok=True)
        return session_path
```

### 12.3.4 TenantAwareSessionManager (租户感知会话管理器)

```python
# session/manager.py
from dataclasses import dataclass
from datetime import datetime
from typing import Optional
import uuid

@dataclass
class TenantSessionEntry:
    """租户隔离的会话条目"""
    session_id: str
    session_key: str
    tenant_id: str
    tenant_type: str
    agent_id: str
    main_key: str
    
    # 会话文件路径 (符合文件管理规范)
    session_file: str  # e.g., "/{tenantId}_private/session-{id}/short-memory.jsonl"
    
    # 元数据
    created_at: datetime
    updated_at: datetime
    provider_override: Optional[str] = None
    model_override: Optional[str] = None
    thinking_level: Optional[str] = None


class TenantAwareSessionManager:
    """
    租户感知的会话管理器
    
    Session Key 格式: tenant:{tenantId}:agent:{agentId}:{mainKey}
    """
    
    SESSION_KEY_PATTERN = "tenant:{tenant_id}:agent:{agent_id}:{main_key}"
    
    def __init__(self, workspace_manager: TenantWorkspaceManager):
        self.workspace_mgr = workspace_manager
        self._sessions: dict[str, TenantSessionEntry] = {}
    
    def derive_session_key(
        self,
        tenant_id: str,
        agent_id: str,
        main_key: str = "main"
    ) -> str:
        """派生带租户信息的 session key"""
        return self.SESSION_KEY_PATTERN.format(
            tenant_id=tenant_id,
            agent_id=agent_id,
            main_key=main_key
        )
    
    def parse_session_key(self, session_key: str) -> dict:
        """解析 session key"""
        parts = session_key.split(":")
        if len(parts) >= 6 and parts[0] == "tenant" and parts[2] == "agent":
            return {
                "tenant_id": parts[1],
                "agent_id": parts[3],
                "main_key": ":".join(parts[4:])  # 支持 mainKey 中包含 ":"
            }
        
        # 旧格式兼容: agent:{agentId}:{mainKey}
        if len(parts) >= 2 and parts[0] == "agent":
            return {
                "tenant_id": "default",
                "agent_id": parts[1],
                "main_key": ":".join(parts[2:]) if len(parts) > 2 else "main"
            }
        
        raise ValueError(f"Invalid session key format: {session_key}")
    
    async def create_session(
        self,
        tenant_context: TenantContext,
        agent_id: str,
        main_key: str = "main"
    ) -> TenantSessionEntry:
        """创建新会话"""
        
        session_id = str(uuid.uuid4())
        session_key = self.derive_session_key(
            tenant_context.tenant_id,
            agent_id,
            main_key
        )
        
        # 确定会话文件路径 (符合文件管理规范)
        if tenant_context.tenant_type == "user_private":
            session_file = (
                f"/{tenant_context.tenant_id}_private/"
                f"session-{session_id}/short-memory.jsonl"
            )
            # 确保目录存在
            await self.workspace_mgr.ensure_session_structure(
                tenant_context.tenant_id,
                session_id
            )
        else:
            # public / user_public 类型不持久化到私有空间
            session_file = None
        
        entry = TenantSessionEntry(
            session_id=session_id,
            session_key=session_key,
            tenant_id=tenant_context.tenant_id,
            tenant_type=tenant_context.tenant_type,
            agent_id=agent_id,
            main_key=main_key,
            session_file=session_file,
            created_at=datetime.utcnow(),
            updated_at=datetime.utcnow()
        )
        
        self._sessions[session_key] = entry
        return entry
    
    async def get_or_create_session(
        self,
        tenant_context: TenantContext,
        session_key: Optional[str] = None
    ) -> TenantSessionEntry:
        """获取或创建会话"""
        
        if session_key:
            # 解析 session key 获取 tenant_id
            parsed = self.parse_session_key(session_key)
            # 使用解析出的 tenant_id，但保持当前请求的 tenant_context
            # (不验证权限，仅用于工作空间隔离)
            
            if session_key in self._sessions:
                return self._sessions[session_key]
        
        # 创建新会话
        return await self.create_session(
            tenant_context,
            tenant_context.agent_id or "default"
        )
```

## 12.4 与 DeepAgents 集成

### 12.4.1 TenantAwareFileBackend (租户感知文件后端)

```python
# backends/tenant_file.py
from deepagents.backends.composite import CompositeBackend
from deepagents.backends.store import StoreBackend
from deepagents.backends.state import StateBackend

class TenantAwareFileBackend:
    """
    租户感知的文件后端 - 使用 DeepAgents CompositeBackend
    
    自动根据路径前缀路由到对应的 StoreBackend:
    - /public/* → PublicStore (namespace: ("app", "public"))
    - /{tenantId}_public/* → UserPublicStore (namespace: ("user", tenantId, "public"))
    - /{tenantId}_private/* → UserPrivateStore (namespace: ("user", tenantId, "private"))
    """
    
    def __init__(self, runtime, tenant_context: TenantContext):
        self.ctx = tenant_context
        self.runtime = runtime
        
        # 三个根目录对应的 backend
        self._init_backends()
        
        # CompositeBackend 路由
        self.backend = CompositeBackend(
            default=self.public_backend,
            routes={
                f"/{tenant_context.tenant_id}_public/": self.user_public_backend,
                f"/{tenant_context.tenant_id}_private/": self.user_private_backend,
            }
        )
    
    def _init_backends(self):
        """初始化三个 StoreBackend"""
        
        # 1. Public - 应用级，使用 StateBackend (会话生命周期)
        self.public_backend = StateBackend(self.runtime)
        
        # 2. User Public - 租户级共享，使用 StoreBackend (持久化)
        self.user_public_backend = StoreBackend(
            self.runtime,
            namespace=lambda ctx: (
                "tenant", self.ctx.tenant_id, "public"
            )
        )
        
        # 3. User Private - 租户私有，使用 StoreBackend (持久化)
        self.user_private_backend = StoreBackend(
            self.runtime,
            namespace=lambda ctx: (
                "tenant", self.ctx.tenant_id, "private"
            )
        )
    
    async def read(self, path: str, **kwargs) -> str:
        """读取文件 - 自动路由"""
        return await self.backend.aread(path, **kwargs)
    
    async def write(self, path: str, content: str, **kwargs):
        """写入文件 - 自动路由"""
        return await self.backend.awrite(path, content, **kwargs)
    
    async def list_files(self, path: str = "/") -> list:
        """列出文件"""
        return await self.backend.als_info(path)
    
    def get_session_file_path(self) -> str:
        """获取当前会话的 short-memory.jsonl 路径"""
        if self.ctx.tenant_type == "user_private" and self.ctx.session_id:
            return f"/{self.ctx.tenant_id}_private/session-{self.ctx.session_id}/short-memory.jsonl"
        return None
```

```python
    agent_config: AgentConfig,
    runtime: ToolRuntime
) -> CompiledStateGraph:
    """
    创建租户感知的 Agent
    
    关键: 注入租户感知的 FileBackend
    """
    
    # 1. 创建租户感知的文件后端
    file_backend = TenantAwareFileBackend(runtime, tenant_context)
    
    # 2. 构建系统提示词 (包含租户信息)
    builder = SystemPromptBuilder(
        workspace_dir=tenant_context.get_workspace_path("session"),
        config=SystemPromptConfig(
            prompt_mode=agent_config.system_prompt_mode,
            bootstrap_max_chars=agent_config.bootstrap_max_chars,
            enable_skills=agent_config.enable_skills,
        )
    )
    
    system_prompt = builder.build(
        agent_id=agent_config.id,
        model=agent_config.model,
        tenant_id=tenant_context.tenant_id  # 添加租户信息
    )
    
    # 3. 创建 DeepAgents Agent
    agent = create_deep_agent(
        model=init_chat_model(agent_config.model),
        system_prompt=system_prompt,
        tools=agent_config.tools,
        # 注入租户感知的 backend
        backend=file_backend
    )
    
    return agent
```

## 12.6 关键设计决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| **租户标识** | X-Tenant-ID Header / Session Key | 简单直接，无权限验证开销 |
| **Session Key** | `tenant:{tenantId}:agent:{agentId}:{mainKey}` | 向后兼容，增加租户维度 |
| **存储隔离** | DeepAgents Namespace | 复用现有基础设施，路径即隔离 |
| **文件路由** | CompositeBackend 路径前缀 | 符合文件管理规范的三根目录设计 |
| **会话存储** | 租户隔离的 JSONL | 符合四层结构，便于调试 |
| **权限验证** | 无 | 仅通过工作空间路径隔离，简化设计 |
