## 架构兼容性说明

### Deep Agents 与 worker.py 的接口映射

```
worker.py                          Deep Agents (CompiledStateGraph)
─────────────────                  ───────────────────────────────
graph_input = {                    → AgentState.messages 接收 messages
  "messages": [...],                  其余字段被 LangGraph 静默忽略
  "plan": [],                      
  "todos": [],                     
  ...30+ legacy fields             
}

astream_events(input, version="v2") → 兼容，Deep Agents 使用相同接口
aget_state(config)                  → 兼容，返回 StateSnapshot
snapshot.interrupts                 → 兼容，Deep Agents via HumanInTheLoopMiddleware
Command(resume=value)               → 兼容，LangGraph 原生 Resume 机制
```

### 关键结论

Deep Agents 编译图是标准的 LangGraph `CompiledStateGraph`，worker.py 的调用方式（astream_events + aget_state）是 LangGraph 标准接口，**完全兼容**。
