### 核心功能实现

#### 知识注入机制
```python
# 在每个问题时注入（awrap_model_call）
state = request.runtime.state
if state.get("_ontology_injected", False):
    return await handler(request)  # 已注入，跳过

# 精细化过滤（预留接口）
relevant_objects = self._filter_relevant_objects(user_query, self.mounted_objects)

# 构建并注入 Schema
ontology_schema = await self._build_ontology_schema(relevant_objects)
updated_request = request.override(system_message=append_to_system_message(...))

# 标记已注入
state["_ontology_injected"] = True
```

#### 工具类型分离
```python
# Worker 端分离逻辑
mounted_objects = []  # OBJECT/VIEW
other_tools = {}      # AGENT/FUNCTION

for tool_name, tool_meta in tools_dict.items():
    if tool_type in ("OBJECT", "VIEW"):
        mounted_objects.append(tool_code)
    else:
        other_tools[tool_name] = tool_meta.get("tool_func")
```
