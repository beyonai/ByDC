# G01 Worker闲聊短路

## 1. 目标
在worker入口增加轻量闲聊短路，命中后直接回复，不进入LangGraph。

## 2. 范围
worker.py入口判定与短路回复；保留ext_command优先级。

## 3. 依赖
无

## 4. 并行性
可并行

## 5. 验收标准
1) 闲聊命中时图节点执行计数为0；2) 非闲聊请求行为不变；3) 有单测覆盖中英文闲聊样例。

1. 变更必须包含对应单元测试或回归测试。
2. 若涉及中断/恢复，必须带 checkpoint 相关字段验证。
