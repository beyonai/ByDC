# 1. 项目概述 - SDK 版本

## 1.1 目标
基于 deepagents 框架构建一个最小化的 OpenClaw Gateway **Python SDK**，核心功能包括：
- **SDK 级 API**: 提供 Python 库而非独立服务
- 用户与 Agent 的双向通信
- 多 Agent 切换
- 斜杠命令支持（`/model`, `/reset`, `/help` 等）
- **无 WebSocket/HTTP 服务器**: 纯进程内调用

## 1.2 技术栈
- **框架**: deepagents (LangGraph-based)
- **语言**: Python 3.11+
- **持久化**: 本地文件系统 (JSON/JSONL)
- **API 风格**: 异步优先 (async/await)，支持同步包装器

## 1.3 架构变更说明

### 原设计 vs SDK 设计

| 方面 | 原设计 (服务器) | 新设计 (SDK) |
|------|----------------|--------------|
| **对外接口** | WebSocket/HTTP 服务器 | Python API |
| **使用方式** | 独立进程，网络通信 | 库嵌入，函数调用 |
| **事件流** | WebSocket 帧 | Python 异步迭代器/回调 |
| **适用场景** | 多客户端远程服务 | 单应用内集成 |

### 剥离的组件
- ❌ WebSocket Server
- ❌ HTTP API 端点
- ❌ 协议握手 (connect/hello-ok)
- ❌ 传输层认证

### 新增的组件
- ✅ GatewayClient (高级 API)
- ✅ EventEmitter (事件系统)
- ✅ 流式响应迭代器
