# 2. 核心架构对比

## 2.1 OpenClaw Gateway 架构（Node.js + Pi SDK）

```
┌─────────────────────────────────────────────────────────┐
│  Gateway (Node.js + Pi SDK)                             │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐     │
│  │  WebSocket  │  │  Session    │  │   Agent     │     │
│  │  Server     │  │  Manager    │  │   Runner    │     │
│  └──────┬──────┘  └──────┬──────┘  └──────┬──────┘     │
│         │                │                │            │
│  ┌──────┴────────────────┴────────────────┴──────┐     │
│  │           Pi SDK (Embedded)                    │     │
│  │  - createAgentSession()                        │     │
│  │  - AgentSession.prompt()                       │     │
│  │  - Tool execution loop                         │     │
│  └────────────────────────────────────────────────┘     │
└─────────────────────────────────────────────────────────┘
```

## 2.2 Python 实现架构 - SDK 版本 (deepagents)

```
┌─────────────────────────────────────────────────────────┐
│  OpenClaw Gateway SDK (Python + deepagents)             │
│  ┌─────────────────────────────────────────────────┐   │
│  │  SDK API Layer (用户直接调用)                      │   │
│  │  ┌─────────────┐  ┌─────────────┐              │   │
│  │  │ GatewayClient │  │  EventEmitter │              │   │
│  │  │ (高级API)     │  │  (事件系统)    │              │   │
│  │  └─────────────┘  └─────────────┘              │   │
│  └─────────────────────────────────────────────────┘   │
│                         │                               │
│  ┌──────────────────────┼──────────────────────────┐   │
│  │                      ▼                          │   │
│  │  ┌─────────────┐  ┌─────────────┐  ┌─────────┐ │   │
│  │  │  Session    │  │   Agent     │  │ Command │ │   │
│  │  │  Manager    │  │   Registry  │  │ Router  │ │   │
│  │  └──────┬──────┘  └──────┬──────┘  └────┬────┘ │   │
│  │         │                │               │      │   │
│  │  ┌──────┴────────────────┴───────────────┘      │   │
│  │  │           Message Queue System               │   │
│  │  │  ┌─────────┐  ┌─────────┐  ┌─────────┐      │   │
│  │  │  │ Enqueue │  │  Drain  │  │ Dedupe  │      │   │
│  │  │  │ Handler │  │ Handler │  │  Cache  │      │   │
│  │  │  └─────────┘  └─────────┘  └─────────┘      │   │
│  │  └─────────────────────────────────────────────┘   │
│  │                      │                            │   │
│  │  ┌───────────────────┴────────────────────────┐   │   │
│  │  │           deepagents SDK                    │   │   │
│  │  │  - create_deep_agent()                      │   │   │
│  │  │  - CompiledStateGraph.invoke()              │   │   │
│  │  │  - Tool calling + Streaming                 │   │   │
│  │  └─────────────────────────────────────────────┘   │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────┘

关键变更:
- 移除 WebSocket/HTTP Server 层
- 新增 SDK API Layer 作为用户直接调用的接口
- EventEmitter 替代 WebSocket 事件流
- 支持同步/异步两种调用方式
```
